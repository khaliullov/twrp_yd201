# Installation

1.  copy this folder to the omni/device/yota/yd201 folder.
2.  . build/envsetup.sh
3.  lunch omni_yd201-userdebug
4.  make -j5 recoveryimage

## Known problems

1.  Touchscreen does not work at all.
